# jwt_app
This is an API only app to demonstrate the basic flow of JWT.
# Required Tools
  1) Ruby 3.1
  2) Rails 7.0
  3) Postman
# Gems
1) bcrypt
2) jwt -> https://github.com/jwt/ruby-jwt
# Setup
  1. Clone the application.
  2. bundle install
  3. rails db:create
  4. rails db:migrate
  5. rails db:seed
